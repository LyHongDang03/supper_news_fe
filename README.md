## Supper News FE

Base project structure for a Next.js App Router frontend.

## Getting started

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` to verify the app is running.

## Current structure

- `src/app`: routing entrypoints
- `src/components`: shared UI/components
- `src/constants`: app constants
- `src/hooks`: reusable hooks
- `src/schemas`: validation schemas
- `src/services`: API/services layer
- `src/stores`: client state stores
- `src/types`: shared TypeScript types
- `src/utils`: utility helpers
- `public`: static assets

Sample auth/demo code has been removed so the repository stays as a clean starter base.
